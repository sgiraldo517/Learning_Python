
# Quick Cool Python Stuff

## venv 

To create virtual environments
```
alias cvenv=python -m venv venv
```

To activate virtual environments
```
alias avenv=venv\Scripts\activate
```

To deactivate virtual environments
```
alias dvenv=deactivate
```

To update the pip on a virtual environment
```
alias upipvenv=python -m pip install --upgrade pip
```
